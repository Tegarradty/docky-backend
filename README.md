# 🚀 DOCKY API - Backend Service

Repositori ini berisi *source code backend* untuk aplikasi **DOCKY**. Dibangun menggunakan Node.js, Express, dan MongoDB, aplikasi ini menyediakan layanan manajemen dokumen dan fitur unggulan **Live Code Execution** yang diamankan di dalam *sandbox* Docker.

## 🛠️ Tech Stack
- **Framework:** Node.js + Express.js
- **Database:** MongoDB
- **Real-time:** Socket.io (WebSocket)
- **Containerization:** Docker & Docker Compose
- **Security:** Helmet, CORS, JWT Auth, Express Rate Limit, Bcrypt

---

## ⚙️ Cara Menjalankan di Komputer Lokal (Untuk Tim)
